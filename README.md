# Come With Me

Come With Me is a mobile-first app that turns your saved TikTok and Instagram posts into a personal guide for real-world experiences -- restaurants, travel, events, and more.

Most people save content and never act on it. Come With Me closes the gap between discovery and doing, with a creator monetization layer that lets guides earn when their recommendations convert.

## Built With
- Next.js + TypeScript
- Tailwind CSS
- Supabase
- Vercel
